# Query: ستحقاق القسط

- Type: select

## Depends On

- [[tables/الاقساط]]

```sql
SELECT Month([تاريخ القسط]) AS [شهر القسط], الاقساط.[الرقم الوظيفي], الاقساط.[مبلغ القسط], الاقساط.[حالة الدفع]
FROM الاقساط
WHERE (((Month([تاريخ القسط]))=Month(Date())) AND ((الاقساط.[حالة الدفع])="لم يتم الدفع"));

```
