# Query: مخالفات على المركبة

- Type: select

## Depends On

- [[tables/المخالفات]]
- [[tables/المركبات]]

```sql
SELECT المركبات.[كود المركبة], المركبات.[نوع المركبة], المركبات.الموديل, المركبات.اللون, المركبات.[رقم اللوحة], المخالفات.المخالفة, المخالفات.[قيمة المخالفة], المخالفات.[حالة الدفع]
FROM المركبات INNER JOIN المخالفات ON المركبات.[رقم اللوحة] = المخالفات.[رقم اللوحة]
WHERE (((المخالفات.[حالة الدفع])=True));

```
