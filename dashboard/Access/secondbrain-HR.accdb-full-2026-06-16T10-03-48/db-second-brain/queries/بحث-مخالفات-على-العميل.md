# Query: بحث مخالفات على العميل

- Type: select

## Depends On

- [[tables/المخالفات]]
- [[tables/بيانات]]

```sql
SELECT [بيانات العميل].[كود العميل], [بيانات العميل].[الاسم العميل], [بيانات العميل].العنوان, [بيانات العميل].[رقم الهاتف العميل], المخالفات.المخالفة, المخالفات.[كود المخالفة], المخالفات.[قيمة المخالفة], المخالفات.[تاريخ المخالفة], المخالفات.[وقت المخالفة], المخالفات.[مكان المخالفة], المخالفات.[حالة الدفع]
FROM [بيانات العميل] INNER JOIN المخالفات ON [بيانات العميل].[الاسم العميل] = المخالفات.[الاسم العميل];

```
