# Query: الشهري

- Type: select

## Depends On

- None

```sql
SELECT [بيان الاجازات].[رقم الوظيفي], Sum([بيان الاجازات].[عدد ايام  الاجازة]) AS [Sumمنعدد ايام  الاجازة]
FROM [بيان الاجازات]
WHERE (((Month([تاريخ بدء الاجازة]) & "/" & Year([تاريخ بدء الاجازة]))=Month(Date()) & "/" & Year(Date())))
GROUP BY [بيان الاجازات].[رقم الوظيفي]
HAVING ((([بيان الاجازات].[رقم الوظيفي])=[Forms]![بيان الاجازات]![رقم الوظيفي]));

```
