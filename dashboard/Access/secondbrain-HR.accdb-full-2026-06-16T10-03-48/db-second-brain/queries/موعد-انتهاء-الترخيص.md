# Query: موعد انتهاء الترخيص

- Type: select

## Depends On

```sql
SELECT المركبات.[كود المركبة], المركبات.[نوع المركبة], المركبات.اللون, المركبات.[رقم اللوحة], المركبات.[مرخصة لغاية]
FROM المركبات
WHERE (((المركبات.[مرخصة لغاية])>=Date()-100 And (المركبات.[مرخصة لغاية])<=Date()+60));
```
