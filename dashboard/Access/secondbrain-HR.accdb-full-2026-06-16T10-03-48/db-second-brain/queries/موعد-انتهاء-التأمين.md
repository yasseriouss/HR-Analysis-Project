# Query: موعد انتهاء التأمين

- Type: select

## Depends On

- [[tables/المركبات]]

```sql
SELECT المركبات.[كود المركبة], المركبات.[نوع المركبة], المركبات.الموديل, المركبات.اللون, المركبات.[رقم اللوحة], المركبات.[تاريخ انتهاء التأمين], المركبات.[شركة التأمين]
FROM المركبات
WHERE (((المركبات.[تاريخ انتهاء التأمين])>=Date()-100 And (المركبات.[تاريخ انتهاء التأمين])<=Date()+60));

```
