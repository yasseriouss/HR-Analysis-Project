# Query: تنبيه بالمخالفات

- Type: select

## Depends On

- [[tables/المخالفات]]
- [[tables/المركبات]]

```sql
SELECT المخالفات.المخالفة, المخالفات.[قيمة المخالفة], المخالفات.[حالة الدفع], [بيانات العميل].[الاسم العميل], [بيانات العميل].[رقم الهاتف العميل], المركبات.[نوع المركبة], المركبات.[رقم اللوحة]
FROM المركبات INNER JOIN ([بيانات العميل] INNER JOIN المخالفات ON [بيانات العميل].[الاسم العميل] = المخالفات.[الاسم العميل]) ON المركبات.[رقم اللوحة] = المخالفات.[رقم اللوحة]
WHERE (((المخالفات.[قيمة المخالفة])>=1 And (المخالفات.[قيمة المخالفة])<=100000));

```
